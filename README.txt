Project Code Submission

This folder contains all code used in the project.

Structure:
- preprocessing/: data processing scripts
- baselines/: TF-IDF and random baseline
- embeddings/: Word2Vec and FinBERT models
- analysis/: lexical cue analysis
- notebooks/: HAN and fine-tuning experiments

Main files:
- tfidf_logistic_large.py
- word2vec_logistic_large.py
- finbert_embedding_large.py
- han_model.ipynb

Notebooks include corresponding HTML files.
